doGet method responding to GET calls. According to different request types, executes corresponding processes and returns the resulting data.

Accepts two parameters from the GET method:

  1. type : Request type. May be i. queryData ii. insertData iii. listData
  2. input : Input data of a request. Each request type requires different input: queryData : A search term such as name of a national park. insertData : List of checkbox id's as a string. listData : No input

Queries data from wikidata and returns the resulting data as a String in an internal data format.

Internal data format is as follows: ROW&&ROW&&ROW&&...&&ROW

Each ROW is formatted as follows: COLUMN||COLUMN||...||COLUMN
